title: "Создание PWA"
---

Для создания PWA приложения после входа в личный кабинет, Вам нужно зайти в раздел Дашборд с левой стороны.

Справа находим \+ **Создать PWA**

<Frame>
  ![Image](/images/image-15.png)
</Frame>

Для создания PWA приложения Вам необходимо привязать собственный домен.

Для того, чтобы использовать собственный домен, нужно п

1. Передать управление DNS записями вашего домена в аккаунт Cloudflare.
2. Прописать ниже домен, Cloudflare e-mail, Cloudflare API ключ.

Далее нажимаем кнопку "Проверить домен" и ждем пока система проверит настройки.

<Frame>
  ![Image](/images/image-16.png)
</Frame>

С инструкцией по передаче домена под управление CloudFlare, можно ознакомиться здесь.

Дождитесь обновления DNS записей. Это может занять некоторое время (от нескольких минут до 24 часов)

<Frame>
  ![Image](/images/image-17.png)
</Frame>

Нажимаем кнопку **Сохранить** и переходим **Далее**.

Далее нужно ввести ссылку на оффер, куда должны попадать пользователи PWA приложения.

<Frame>
  ![Image](/images/image-18.png)
</Frame>

После ввода ссылки выбираем устройства на которых будет показываться PWA.

Если хотим оставить только устройства Andrid, то кликаем справа как указано стрелкой. Тогда при включении PWA показывается только на Android

<Frame>
  ![Image](/images/image-19.png)
</Frame>

Выбираем куда отправлять IOS трафик

- показывать белую страницу
- показывать сразу оффер
- на страницу установки PWA
- на URL

Выбираем куда отправлять десктопный трафик

- показывать белую страницу
- показывать сразу оффер
- на страницу установки PWA
- на URL

Нажимаем **Далее.**

Во вкладке оформление нужно выбрать тип страницы установки:

- Нативный стор
- Playable

В настройках параметров нужно указать название приложения и слева загрузить иконку.

<Frame>
  ![Image](/images/image-28.png)
</Frame>

Далее можете воспользоваться галереей и выбрать подходящий Playable.

<Frame>
  ![Image](/images/image-29.png)

Далее нужно выбрать тему инструкции по установке и модалки push-уведомлений на iOS.

- авто
- светлая
- темная
  <Frame>
    ![Image](/images/image-30.png)
  </Frame>

## **Аналитика**

Для того, чтобы собрать данные таких как **Регистрации/** **Депозит**, нужно их передать постбеками из:

- Вашей Партнерки
- Трекера (Keitaro, Binom)

Смотри вкладку Интеграция с Keitaro

### **S2S Постбэки**

Подключите S2S постбэки для автоматического отслеживания конверсий из вашего трекера или партнерской программы. Скопируйте нужный постбэк-URL в настройки целевого события — события (регистрация, депозит и др.) будут автоматически записываться в статистику.

Постбэк для регистраций

[https://api.pwa-farm.tech/knocks/?external\_id={external\_id}&event=registration](https://api.pwa-farm.tech/knocks/?external_id={external_id}&event=registration)

Постбэк для депозитов

[https://api.pwa-farm.tech/knocks/?external\_id={external\_id}&event=deposit&value={profit}&currency=usd](https://api.pwa-farm.tech/knocks/?external_id={external_id}&event=deposit&value={profit}&currency=usd)
