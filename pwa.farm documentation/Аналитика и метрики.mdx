Во вкладке **Аналитика** Вы можете отслеживать подробную статистику по вашим заливам.

После перехода во вкладку **Аналитика**, в левой части можно выбрать, отследить статистику конкретного приложения PWA, либо общую статистику по всем приложениям.

- Динамика метрик
- Трафик/конверсия

Раздел **Динамика метрик** представляет изменения ключевых показателей за выбранный период в виде линейного графика. Он позволяет отслеживать тенденции, сравнивать показатели между собой и оценивать эффективность трафика во времени.

Содержит график со следующими метриками:

- **Уники** - количество кликов уникальных пользователей (повторные клики от одного и того же пользователя не учитываются на графике)
- **Инсталлы** - фактические установки PWA приложений
- **Открытия** - количество первых открытий PWA прилржений
- **Регистрации** - количество регистраций. Данные о регистрациях мы получаем на основе постбеков, которые передает нам ваша Партнерка или Трекер.
- **Депозиты** - количество депозитов пользователей. Данные о депозитах мы получаем на основе постбеков, которые передает нам ваша Партнерка или Трекер.

Каждая метрика отображается отдельной линией, что позволяет наглядно сравнить их динамику и выявить изменения на каждом этапе воронки привлечения пользователей. Раздел помогает быстро оценить эффективность рекламной кампании, обнаружить рост или снижение ключевых показателей и проанализировать результаты за выбранный период.

Раздел **Трафик/конверсия** содержит таблицу со следующими метриками за конкретную дату:

- **Уники**
- **Инсталлы**
- **Открытия**
- **Регистрации**
- **Депозиты**

А также конверсии

- **visit/inst -** процент посетителей, которые установили приложение. Чем выше показатель, тем эффективнее страница перехода и процесс установки.
- **inst/reg -** процент пользователей, установивших приложение, которые затем зарегистрировались. Показывает эффективность регистрации после установки.
- **reg/dep -** процент зарегистрированных пользователей, совершивших первый депозит. Это показатель того, насколько успешно регистрация конвертируется в платящего пользователя.

Таким образом, эти метрики позволяют оценить эффективность каждого этапа воронки: **открытие → установка → регистрация → депозит**.

Раздел **Гео-аналитика** отображает распределение трафика и ключевых показателей эффективности по странам. Он позволяет определить, из каких регионов приходит аудитория и сравнить результаты по каждому гео.

Для каждой страны отображаются следующие метрики:

- **Страна** — страна, из которой пришли пользователи.
- **Уники** — количество уникальных пользователей из выбранной страны.
- **Инсталлы** — количество пользователей, установивших приложение.
- **Открытия** — количество первых запусков приложения после установки.
- **Регистрации** — количество пользователей, успешно зарегистрировавшихся в приложении.
- **Депозиты** — количество пользователей, совершивших первый депозит.
- **CPI (Cost Per Install)** — средняя стоимость привлечения одной установки в выбранной стране. Рассчитывается как отношение затрат на рекламу к количеству полученных установок.

Раздел помогает сравнить эффективность трафика по разным странам, определить наиболее перспективные гео и оценить стоимость привлечения пользователей в каждом регионе.
