Вкладка Трекинг находится по левой стороне главного меню.

<Frame>
  ![Image](/images/image-34.png)
</Frame>

Раздел **«Трекинг»** предназначен для управления рекламными пикселями **Facebook** и **TikTok**. Подключенные пиксели используются для отслеживания конверсий, анализа эффективности рекламных кампаний и оптимизации таргетинга.

## Просмотр списка пикселей

После открытия раздела отображается список всех добавленных пикселей.

Для каждого пикселя доступна следующая информация:

- **Имя пикселя** — название, указанное при создании.
- **Тип** — рекламная платформа (Facebook или TikTok).
- **Pixel ID** — уникальный идентификатор пикселя.
- **Токен** — токен доступа, используемый для интеграции (если применимо).

Если пиксели еще не добавлены, отображается сообщение **«Данные отсутствуют»**.

## Фильтрация

Для быстрого просмотра можно отобразить пиксели определенной платформы:

- **Все** — отображаются все пиксели.
- **Facebook** — отображаются только пиксели Facebook.
- **TikTok** — отображаются только пиксели TikTok.

## Поиск

Поле поиска позволяет быстро найти нужный пиксель по:

- названию;
- Pixel ID;
- токену.

## Добавление пикселя **Facebook**

Чтобы подключить новый пиксель нажмите кнопку **«Добавить пиксель»**.

<Frame>
  ![Image](/images/image-35.png)
</Frame>

Для подключения нового **пикселя Facebook** выполните следующие действия:

1. В разделе **Трекинг** нажмите кнопку **«Добавить пиксель»**.
2. В блоке **Основные параметры пикселя** выберите тип **Facebook**.
3. Заполните поля:
   - **Пользовательское имя пикселя** — укажите произвольное название, под которым пиксель будет отображаться в системе.
   - **Числовой ID из администратора Facebook** — введите идентификатор пикселя, полученный в Facebook Business Manager.
   - **API Token Facebook Pixel** — укажите API-токен, необходимый для передачи событий через Facebook Conversion API.
4. В блоке **События Facebook** настройте соответствие между событиями приложения (PWA) и событиями Facebook. Для каждого события выберите необходимое значение из выпадающего списка:
   | Событие в PWA | Событие Facebook |
   | :-- | :-- |
   | Открытие страницы | ViewContent |
   | Установка | Lead |
   | Регистрация | Complete Registration |
   | Депозит | Purchase |
   | При необходимости значения можно изменить, выбрав другое событие из списка. |  |
5. После заполнения всех обязательных полей нажмите **«Сохранить»**.
6. После успешного сохранения новый пиксель появится в разделе **Трекинг** и начнет использоваться для отправки выбранных событий в Facebook.

## Добавление пикселя TikTok

Для подключения нового пикселя TikTok выполните следующие действия:

1. В разделе **Трекинг** нажмите кнопку **«Добавить пиксель»**.
2. В блоке **Основные параметры пикселя** выберите тип **TikTok**.
3. Заполните обязательные поля:
   - **Пользовательское имя пикселя** — укажите название, под которым пиксель будет отображаться в системе.
   - **TikTok Pixel ID** — введите идентификатор пикселя, полученный в TikTok Ads Manager.
   - **Access Token** — укажите токен доступа, необходимый для интеграции с TikTok Business API.
4. В блоке **События TikTok** настройте соответствие между событиями приложения (PWA) и событиями TikTok. Для каждого события PWA выберите событие TikTok из выпадающего списка:
   | Событие в PWA | Событие TikTok |
   | :-- | :-- |
   | Открытие страницы | ViewContent |
   | Установка | Download |
   | Регистрация | Complete Registration |
   | Депозит | Purchase |
   | При необходимости выбранные события можно изменить, выбрав другие значения из выпадающих списков. |  |
5. После заполнения всех обязательных полей нажмите **«Сохранить»**.
6. После успешного сохранения пиксель появится в разделе **Трекинг** и будет использоваться для передачи настроенных событий в TikTok.
