**Функционал сервиса**

На главной страничке личного кабинета во вкладке **Дашборд** можно :

**1. Просмотр списка PWA**

- Отображается список всех созданных PWA-приложений.
- Для каждого приложения показываются:
  - название;
  - домен;
  - дата создания;
  - текущий статус;
  - действия (редактирование и дополнительное меню).

**2. Фильтрация по статусу**

Вы можете быстро переключаться между категориями:

- **Все** — отображаются все приложения.
- **Активные** — опубликованные и работающие PWA.
- **Черновики** — приложения, которые еще не готовы к публикации.
- **Ожидание NS** — приложения, ожидающие обработки или проверки.
- **С ошибкой** — приложения, требующие внимания из-за возникших ошибок.

Рядом с каждым фильтром отображается количество приложений.

**3. Поиск**

Поле поиска позволяет находить приложение по:

- названию;
- доменному имени.

**4. Управление приложением**

Для каждого PWA доступен переход к редактированию;

**5. Создание нового PWA**

В правом верхнем углу расположена кнопка **«Создать PWA»**, которая запускает процесс создания нового приложения.

Во вкладке **Аналитика** отображается статистика и показатели эффективности выбранного PWA-приложения.

На графике можем увидеть следующие метрики:

- Уники
- Инсталлы
- Открытия
- Регистрации
- Депозиты

Также можем увидеть таблицу Трафик/Конверсии и Гео-аналитику. Подробнее смотри во вкладке **Аналитика.**
